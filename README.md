## Формат входных данных сетки
```py
N # Число вершин
x0 y0 [z0]
x1 y1 [z1]
# ... (N строк)
M # Число внутренних элементов
i1 i2 i3 [i4 ...] m
# ... (M строк)
K # Число граничных элементов
i1 i2 [i3 ...] b
# ... (K строк)
```
Где:
- `xk, yk, zk` - Координаты k-ой вершины;
- `i1, i2, ...` - Индексы вершин элемента;
- `m` - Индекс материала;
- `b` - Индекс граничного условия;
